## Комментарий заказчика (что взять отсюда)

> «Можно логотип посередине сделать так же на чёрном фоне, а по бокам сделать эти фигурки. Можете по-разному их расположить, главное чтобы один край визуально тяжелее другого не был.»

**Вывод для героя нашего сайта:**
1. **Чёрный фон.**
2. **Логотип — по центру** (не сбоку, как в `clustered`).
3. **Фигурки — по бокам / вокруг**, расположение свободное, но **сбалансированное** — ни один край не должен перевешивать визуально (ориентир — вариант `balanced`, а не `clustered`).
4. Сами «фигурки» = это и есть **элементы конструктора заказчика** (в первом примере «Маски мира» их роль играли треугольники).

## Логотип и типографика

- Логотип **«БУМАГУ, ПОЖАЛУЙСТА»** — крупный, белый, в **ретро/groovy (psychedelic) стиле**: округлые «тягучие» буквы, наклон, заглавные.
- Подзаголовок (под логотипом): обычный гротеск, в 2–3 строки — «студия упаковки подарков и бумажного творчества».

## Фигурки (элементы)

Плоские «бумажные» фигуры в стиле вырезанных из бумаги цветов:
- ромашки (жёлтые, с прорезями-лепестками),
- круглые «облачные» цветы (синие),
- 5-лепестковые цветы (малиновые),
- листики/кляксы (лаймовые).

Контуры с тонкими чёрными прорезями «под бумагу». Это визуальный ориентир для «элементов конструктора», которые расставляем вокруг логотипа.
