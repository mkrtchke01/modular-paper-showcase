## 1. Что за заказ

Лендинг для **«Бумагу, пожалуйста»** — студии упаковки подарков и бумажного творчества. Заказ с Kwork.
Полный текст ТЗ: [`ORDER.md`](ORDER.md).

**Нужно сделать сайт**, похожий по **структуре и подаче** на сайт-пример, но:
- в **фирменных цветах** заказчика;
- с **элементами конструктора** заказчика вместо треугольников из примера;
- **без анимации вращения** элементов;
- **обычный (стандартный) курсор**;
- карточки товаров при наведении **немного увеличиваются**;
- разместить на **бесплатном хостинге**, сайт должен прожить **минимум месяц**.

### Разделы (по ТЗ)
1. **«О нас»** — ознакомительный раздел.
2. **Товары** — карточки товаров (hover = лёгкое увеличение).
3. **«Контакты»**.

## 5. Концепт реализации

**Первый экран (Hero):** чёрный фон, **логотип по центру** (`для лого-08.png`) + подзаголовок, **фигурки/элементы конструктора расставлены вокруг сбалансированно** (ни один край не тяжелее). Без вращения. Меню сверху: Каталог/Товары · О нас · Контакты.

**О нас:** короткий рассказ о студии (текст можно написать от себя как плейсхолдер — см. статус).

**Товары:** сетка карточек, при наведении `transform: scale(~1.03)`. Контент товаров заказчик пока не прислал → **плейсхолдеры** (можно опираться на ассортимент упаковочной бумаги).

**Контакты:** блок с реальными контактами выше.

### Технологии и хостинг
- ⚠️ Путь проекта содержит **кириллицу и пробелы** (`OneDrive\Рабочий стол\...`). Сборщики (Vite/Next) на таком пути ломаются (см. memory). → Рекомендуется **статический сайт без сборки** (HTML/CSS/JS) — он же проще всего деплоится и надёжнее живёт на бесплатном хостинге.
- Бесплатный хостинг на ≥1 месяц: **Netlify / Cloudflare Pages / GitHub Pages** (статика, без срока «засыпания»).
