# Заказ (Kwork)

## Описание от заказчика

Нужно сделать сайт, похожий на пример, который показал заказчик, но в его цветах и с его элементами.

## Что нужно сделать

### 1. Общий стиль
- Сайт должен быть похож по структуре и подаче на пример.
- Цвета — наши (заказчика).
- Вместо треугольников из примера использовать **наши элементы конструктора**.
- Анимацию вращения элементов делать **не нужно**.
- Курсор мышки оставить обычный, стандартный.

### 2. Разделы сайта
- Ознакомительный раздел **«О нас»**.
- Раздел с товарами.
- Раздел **«Контакты»**.

### 3. Товары
- Сделать карточки товаров.
- При наведении на карточку она должна **немного увеличиваться**.

### 4. Контактная информация
- Контакты заказчик предоставит отдельно.

### 5. Хостинг
- Подобрать **бесплатный** хостинг.
- Важно, чтобы сайт проработал хотя бы **месяц** (пока не выставят оценки).

- Фирменные цвета.
- «Элементы конструктора» (вместо треугольников) — материалы/исходники.
- Контактные данные.
- Контент для раздела «О нас» и список товаров (фото, названия, описания, цены).
